# WebSocket Implementation

This directory contains the WebSocket protocol implementation.

## Components

- Opening handshake
- Frame parsing and generation
- Message fragmentation
- Ping/pong support
- Close handshake
- Extensions support

## References

- [RFC 6455](https://tools.ietf.org/html/rfc6455) - The WebSocket Protocol

## Extensions
